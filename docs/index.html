Tất Cả Trong Một

Từ UI library đến các module logic phức tạp — mọi thứ đã được module hoá và sẵn sàng dùng.

Custom UI Library

Tab system, toggle, dropdown, slider, input, danh sách cuộn — tất cả với animation tween mượt mà và đầy đủ theme hỗ trợ.

Potato Mode

Tắt particle, shadow, texture, fog, PostEffect và hàng loạt hiệu ứng đồ hoạ theo batch — giảm lag đáng kể mà không cần restart.

Animation Engine

Hệ thống tween đa lớp: mở/đóng UI với icon shrink, hover, drag-to-move, click feedback — mọi trạng thái đều có hiệu ứng.

Search & Highlight

Tìm kiếm realtime trong UI với auto-scroll tới kết quả, highlight nền và viền neon vàng, cache descendants cho hiệu suất cao.

Auto Clicker

Tối đa 20 điểm click kéo-thả được, tốc độ tuỳ chỉnh theo mili giây, hiệu ứng chớp sáng khi click, hỗ trợ cả mouse lẫn touch.

Player Tools

Fly, Noclip, Teleport tới player/chuột/click, Target Lock, ESP outline, Anti-AFK tự động — tất cả có thể toggle realtime.

Hx Build

Chọn, di chuyển, neo và hàn các khối trong workspace. Hỗ trợ nhóm khối, undo, outline vùng chọn trực quan.

Config Manager

Lưu/đọc cấu hình riêng theo từng game và global. Tự động áp dụng theme, phím tắt, kích thước UI khi mở lại.

Tài Liệu Sử Dụng

Toàn bộ API được chia theo từng Module để dễ dàng tra cứu và tích hợp vào dự án của bạn.

Khởi chạy qua Hub trung tâm

Chỉ cần dùng Executor (Delta, Fluxus, Evon,...) thực thi đoạn mã dưới đây. Một Menu UI sẽ hiện ra cho phép chọn các Script đã được viết sẵn trong thư viện ChaosSystem.

LUA
loadstring(game:HttpGet("https://raw.githubusercontent.com/9311-nex4/Hx/main/Hx_Loader.lua"))()

Khởi tạo Cửa sổ & Tab

Load trực tiếp module `ThuVienUI.lua`. Sử dụng `TaoWindow` để dựng khung UI, sau đó gọi `TaoTab` để thêm các trang nội dung độc lập.

LUA
local UI = loadstring(game:HttpGet("https://raw.githubusercontent.com/9311-nex4/Hx/main/Utilities/ThuVienUI.lua"))()

local Window = UI:TaoWindow({ TieuDe = "My Custom Script" })
local Tab1 = Window:TaoTab("Người Chơi")
local Tab2 = Window:TaoTab("Cài Đặt")

Thêm các Elements điều khiển

Gọi các hàm từ đối tượng Tab để sinh ra Button, Toggle, Slider, hoặc Dropdown. Các hàm đều có callback Event trả về giá trị realtime.

LUA
Tab1:TaoButton({
    Ten = "Nhấn Để Dịch Chuyển",
    SuKien = function() print("Đã click!") end
})

Tab1:TaoToggle({ 
    Ten = "Kích hoạt God Mode", 
    SuKien = function(TrangThai) print("God Mode:", TrangThai) end 
})

Tab2:TaoSlider({
    Ten = "Tốc Độ Chạy (WalkSpeed)",
    Min = 16, Max = 100, MacDinh = 16,
    SuKien = function(GiaTri) game.Players.LocalPlayer.Character.Humanoid.WalkSpeed = GiaTri end
})

Tab2:TaoDropdown({
    Ten = "Chọn Mục Tiêu",
    DanhSach = {"Mục Tiêu A", "Mục Tiêu B", "Mục Tiêu C"},
    MacDinh = "Mục Tiêu A",
    SuKien = function(LuaChon) print("Đang chọn:", LuaChon) end
})

Gọi hộp thoại Thông báo

Module `ThongBao.lua` xử lý các Notify nổi mượt mà trên màn hình người dùng. Rất hữu ích để báo cáo trạng thái chức năng.

LUA
local TB = loadstring(game:HttpGet("https://raw.githubusercontent.com/9311-nex4/Hx/main/Utilities/ThongBao.lua"))()

TB.thongbao("Hx Script", "Tính năng đã bật thành công!", 3)

TB.thongbaoloi("Cảnh Báo Lỗi", "Mất kết nối máy chủ hoặc API tạch!")

TB.thongbaoxacnhan("Xác nhận hành động", "Bạn có chắc chắn muốn rời Server?", 
    function() game:Shutdown() end, 
    function() print("Đã hủy!") end
)

Tối ưu Client với ConfigManager

Loại bỏ bóng đổ, sương mù và kích hoạt FullBright để tăng FPS cho máy yếu.

LUA
local Config = loadstring(game:HttpGet("https://raw.githubusercontent.com/9311-nex4/Hx/main/Utilities/MenuConfigManager.lua"))()

Config.ReduceLags(true)
Config.NoShadows(true)
Config.RemoveFog(true)
Config.FullBright(true)

Gọi Utils Logic phổ biến

Module `ETC_Logic.lua` chứa sẵn các Script thông dụng cho dân Exploit như Fly, Noclip hay Click TP.

LUA
local Logic = loadstring(game:HttpGet("https://raw.githubusercontent.com/9311-nex4/Hx/main/Utilities/ETC_Logic.lua"))()

Logic.ToggleFly(true)
Logic.ToggleNoclip(true)
Logic.SetClickTP(true)
